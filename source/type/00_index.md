## 简介

Type 是一个简洁、美观并且力求快速的 Hexo 主题，能以较好的样式展示文字和图片，并且对移动端、夜间模式做了一些优化。希望 Type 能 在 Hexo 上承载您的文字、影像作品。

主题仍在开发中，最新开发进展请看：[aiokr/Hexo-Theme-Type at dev](https://github.com/aiokr/Hexo-Theme-Type/tree/dev)
